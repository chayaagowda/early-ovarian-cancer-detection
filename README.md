


# Early Ovarian Cancer Detection using Machine Learning

## Overview
This project focuses on early detection of ovarian cancer using supervised machine learning techniques. The goal is to improve prediction accuracy and assist in early diagnosis through data-driven analysis.

## Objectives
- Apply machine learning models for classification
- Compare model performance
- Improve early detection accuracy
- Build a research-oriented predictive workflow

## Methodology
1. Data preprocessing and cleaning
2. Feature selection
3. Model training
4. Performance evaluation
5. Comparative analysis

## Models Used
- K-Nearest Neighbors (KNN)
- Decision Tree
- Random Forest

## Results
- Achieved ~89% accuracy
- Random Forest performed best
- Improved precision and recall for early-stage detection

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Research Contribution
This project explores the application of machine learning techniques for early-stage ovarian cancer detection, highlighting the potential of predictive analytics in healthcare research.
